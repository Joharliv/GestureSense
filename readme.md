# GestureSense 🤖✋

GestureSense is an AI-powered real-time sign language recognition system using computer vision and deep learning.

---

## 🚀 Features
- Static gesture recognition (A–Z)
- Motion gestures (J, Z)
- Smart sentence builder
- Real-time camera prediction

---

## 🧠 Tech Stack
- Python (Flask)
- OpenCV
- NumPy
- HTML, CSS, JavaScript
- Machine Learning / Deep Learning
