# GestureSense

AI-powered sign language recognition system using computer vision and deep learning.

## Features
- Static gesture recognition
- Motion gesture recognition (J, Z)
- Smart sentence builder
- Real-time prediction

## Tech Stack
- Python (Flask)
- OpenCV
- TensorFlow
- HTML, CSS, JavaScript

## Run Project

### Backend
cd backend
pip install -r requirements.txt
python app.py

### Frontend
Open index.html
